# Jet Pilot

> https://github.com/unxsist/jet-pilot

Jet-Pilot is UI for Kubernetes.

## Install

### Linux

```sh
curl -fSLO https://github.com/unxsist/jet-pilot/releases/download/v1.33.0/JET.Pilot_1.33.0_amd64.deb
sudo dpkg -i JET.Pilot_1.33.0_amd64.deb
```
